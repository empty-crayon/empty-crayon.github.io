# emptycrayon.github.io

Static personal website source for [empty-crayon.github.io](https://empty-crayon.github.io).

## Local preview

```bash
cd /home/emptycrayon/Documents/Github-Repositories/empty-crayon.github.io
./scripts/local-serve.sh
```

Open: `http://127.0.0.1:4000`

## Deployment

Deployment is handled by GitHub Actions using:
- `.github/workflows/deploy-pages.yml`

Push to `main` to deploy.
